Multi filtering

the search is important in dropdown , the drop down we build only searches by firstName of the datasource but what if we want to search by multiple fields :


in this dropdown you can search by FirstName , LastName and nationalCode , The code is :
//body <input id="dropdownmulti" class="sans" style="width: 30%"/><br> //script $("#dropdownmulti").kendoDropDownList({ filter: "contains", tagMode: "single", dataTextField: "firstName", dataValueField: "id", dataSource: mydatasource , filtering: function(ev){ var filterValue = ev.filter != undefined ? ev.filter.value : ""; ev.preventDefault(); this.dataSource.filter({ logic: "or", filters: [ { field: "firstName", operator: "contains", value: filterValue }, { field: "lastName", operator: "contains", value: filterValue }, { field: "nationalCode", operator: "contains", value: filterValue }, ] }); } });
